# 🕵️‍♀️ ganx

_Let's try and implement a gan using jax_
⚠️ Nothing in this repo is usable yet ⚠️

## 💾  Install the package
```
pip install git+https://github.com/rosshemsley/ganx
```


## 🚂  Train the net
```
$ python -m ganx.cli.train
```

or from a virtualenv with the package installed
```
$ train
```
